HTML Table Tags and Attributes

Tag/Attribute Example Description
<table> <table></table> Defines a table.
<tr> <tr></tr> Defines a row in a table.
<th> <th>Header</th> Defines a header cell in a table.
<td> <td>Data</td> Defines a cell in a table.
colspan <td colspan="2"> Specifies the number of columns a cell should span.
